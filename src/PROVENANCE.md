# Where this code came from

The modules in this `src/` originated in the research repos
([open-problems](https://github.com/queelius/open-problems),
[computational-explorations](https://github.com/queelius/computational-explorations))
and were copied here so this portfolio reproduces every sequence on its own.

This repo is now **self-contained and primary for its own copies**: it evolves
independently of the research repos, which keep their own versions. No sync is
expected or maintained. The table below is attribution, not a contract.

| File | Originated in |
|------|---------------|
| `rado_numbers.py`, `zarankiewicz*.py`, `graph_saturation.py`, `reaction_networks*.py`, `max_tail_*.py`, `finite_map_stats.py` | open-problems `src/` |
| `sandpile_groups/*` | open-problems `discoveries/algebraic-graph-theory/sandpile-groups/` |
| Schur / sum-free generators (S(Z/nZ,k)) | computational-explorations `src/` |
