OEIS Submission Candidate: S(Z/nZ, 2)
=====================================

Maximum size of a subset of the cyclic group Z/nZ that splits into two
sum-free sets.

---------------------------------------------------------------------
NAME
  a(n) is the maximum size of a subset A of the cyclic group Z/nZ such
  that A can be partitioned into two sum-free sets.

DATA (offset 2; n = 2, 3, 4, ...)
  1, 2, 3, 4, 4, 4, 6, 6, 8, 8, 9, 8, 9, 12, 12, 12, 12, 12, 16

OFFSET
  2

COMMENTS
  A subset S of an abelian group is sum-free if there are no x, y, z in S
  (not necessarily distinct) with x + y = z. Equivalently, a(n) is the
  largest m for which the elements of some m-element subset of Z/nZ can be
  2-colored so that neither color class contains a solution of x + y = z
  (mod n).

  a(n) is non-monotone: a(13) = 8 < a(12) = 9.

  Order-invariant property: for every abelian group G of order n the
  analogous quantity (maximum subset of G partitionable into two sum-free
  sets) equals a(n); that is, the value depends only on n and not on the
  group structure. This was verified for all abelian groups of order
  2 <= n <= 20.

  Trivial bounds: s(n) <= a(n) <= n, where s(n) is the maximum size of a
  single sum-free subset of Z/nZ (the k = 1 case).
